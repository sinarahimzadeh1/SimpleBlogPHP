<h1 align="left">Simple PHP Blog</h1>

###

<p align="left">A simple blog for posting content and organizing it into categories.</p>

###

<h2 align="left">About project</h2>

###

<p align="left">Create posts with images, titles, and more — all managed through categories, a database, and an admin panel. The site also includes a basic authentication system with login and logout capabilities.</p>

###

<h2 align="left">Getting Started</h2>

###

<p align="left">You can modify the database name and credentials inside the PDO_Connection.php file.<br>It’s recommended to keep the default database name as php_project.<br><br>If you place the project files inside a sub-folder instead of the root directory (localhost), you can update the projectFolder value inside helpers.php to match your desired folder name.</p>
